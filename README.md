# Topiary

Dialogue Tree Editor inspired by [twinery](http://www.twinery.org) created with React.

![Topiary Screenshot](./public/topiary.png)

### Installation

1.  clone repo
2.  `npm install` dependencies
3.  `npm start` to load react app.

### Dependencies

* material-ui
* file-saver
* gradstop
* prop-types
* react
* react-dom
* react-draggable
* react-redux
* react-router-dom
* react-scripts
* redux
* reselect
